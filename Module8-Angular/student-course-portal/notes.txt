angular.json          - Angular workspace configuration, build and serve settings
tsconfig.json         - Base TypeScript compiler configuration
tsconfig.app.json     - TypeScript config specific to the app build
package.json          - Project dependencies and npm scripts
src/main.ts           - Application entry point, bootstraps the root component
src/app/app.config.ts - Root configuration, registers providers for the app
src/app/app.component.ts - Root component, the starting component of the app
src/index.html        - Main HTML file, contains the app-root tag
